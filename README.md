# xybot

> Algorithmic MIDI command center for OP-XY

## Controls

- **E1** — select page (DRUM / SYNTH / LFO / PERFORM)
- **E2** — param A (density/fx send/tempo/pattern)
- **E3** — param B (filter/swing/vel/fx2/scene)
- **K2** — generate / action (page dependent)
- **K3** — mutate / next scene (page dependent)
- **K1 (hold)** — alternate layer for play/stop and scene control

## Grid

16×8 grid with per-page layouts: voice visualization, parameter scrubbing, and performance controls.

## Requirements

- norns
- grid (16×8)
- MIDI device (OP-XY or similar)

## Install

```
;install https://github.com/jamminstein/xybot
```
